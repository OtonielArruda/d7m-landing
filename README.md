# D7M — Desbravadores dos Sete Mares

Landing page estática construída somente com HTML, CSS e JavaScript. Abra `index.html` em um navegador para executar.

## Publicação

- **GitHub Pages:** envie a pasta ao repositório e ative Pages nas configurações.
- **Cloudflare Pages:** conecte o repositório, escolha `D7M-Landing` como diretório de publicação e não informe comando de build.
- **Netlify:** arraste a pasta para o painel de deploy ou conecte o repositório com `D7M-Landing` como diretório de publicação.

Antes de publicar, confirme o domínio em `index.html`, `robots.txt` e `sitemap.xml`.
